1.6 - Describe and provide analysis on how your chosen technologies have helped you achieve your outcomes. How has Trello helped you plan and how can git and GitHub help with your codes version control?

Html, CSS and javascript are deeply intertwined with website development to the point where it’s hard to have a good looking website without all of those elements working together properly. Html [3] is primarily responsible for the coding of the website where as CSS [4] is more focused on the graphical appearance of the website EG making an image change aspect ratio and more. Finally, there is javascript [5] which can be used for number of things EG detecting the use of an adblocker running which could that force the user to disable it before using the website.

Trello I’ve found has been largely redundant for me as I tend to keep things in mind but overall I see it as a useful tool when there are more than one person working on a project. I’ve mostly used it to remind me of what work I need to when finishing for the day!

Git and Github are very useful tools to push the code I’ve been working on to the cloud so that I can work on it from any machine. This also works as a good redundancy measure to protect against hardware failure and accidental loss of code providing best practice is in place! I've even gone as far to help automate some of the pushes for rapid testing in the live environment (which is why there are so many commits on this repo!) as there are certain instances where a local test won't show up issues that a live environment will.
