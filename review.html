I firmly believe the role of IT in the modern business place is to streamline things and open up new opportunities that otherwise wouldn’t be present to the companies that invest in the sector. 1.6 - Describe and provide analysis on how your chosen technologies have helped you achieve your outcomes. How has Trello helped you plan and how can git and GitHub help with your codes version control?

Html, CSS and javascript are deeply intertwined with website development to the point where it’s hard to have a good looking website without all of those elements working together properly. Html [3] is primarily responsible for the coding of the website where as CSS [4] is more focused on the graphical appearance of the website EG making an image change aspect ratio and more. Finally, there is javascript [5] which can be used for number of things EG detecting the use of an adblocker running which could that force the user to disable it before using the website.

Trello I’ve found has been largely redundant for me as I tend to keep things in mind but overall I see it as a useful tool when there are more than one person working on a project. I’ve mostly used it to remind me of what work I need to when finishing for the day!

Git and Github are very useful tools to push the code I’ve been working on to the cloud so that I can work on it from any machine. This also works as a good redundancy measure to protect against hardware failure and accidental loss of code providing best practice is in place!

2.1 - Review the ongoing use of IT tools and change approach as needed – in making your own website, are you happy with the tools you used? E.g. Visual Studio Code, Codepen, Trello

I’m very happy with the Visual Studio Code. Before starting to build the website I asked if it would be ok to other another IDE (integrated development environment) and while the answer was yes, based on my own research VSC is the best all-around solution for actually understanding the code involved rather than just making the website for you using a graphical user interface.

VSC in particular is extremely versatile with the ability to change programming languages on the fly, extensions from its own appstore, easy integration with github it truly stands out amongst the crowed and last but not least, the quality of life changes that make typing code much quicker.

Trello has been somewhat useful but as I mentioned before I don’t consider it that great of a tool for a solo project of this scale. I can certainly see the use for larger solo projects but for the sake of building a basic website it’s not needed for me.
